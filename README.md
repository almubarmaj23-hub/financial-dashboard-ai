## ✨ المميزات

| المكون | الوصف |
|--------|-------|
| 📊 **KPI Cards** | 4 بطاقات مؤشرات مالية مع نسب التغيير ومؤشر تقدم |
| 📈 **Revenue Chart** | رسم بياني منطقة مع فلتر ربع/نصف/سنوي + خط الميزانية |
| 🍩 **Expense Pie** | مخطط دائري توزيع المصاريف حسب الفئة |
| 💸 **Cash Flow** | رسم أعمدة للتدفق النقدي الأسبوعي |
| 🔮 **AI Forecast** | توقعات الإيرادات بالذكاء الاصطناعي (دقة 94.2%) |
| 🛡️ **Health Metrics** | مخطط Radar للصحة المالية + شرائط التقدم |
| 📋 **Transactions** | جدول المعاملات مع فلتر وحالة الدفع |
| 🤖 **AI Insights** | رؤى تحليلية تلقائية مع زر التحديث |

## 🛠️ تقنيات البناء

- **Next.js 14** — Static Export لـ GitHub Pages
- **TypeScript** — أمان برمجي كامل
- **TailwindCSS** — تصميم Glass Morphism داكن
- **Recharts** — رسوم بيانية تفاعلية (Area, Pie, Bar, Radar, Composed)
- **Lucide React** — أيقونات SVG حديثة
- **Cairo Font** — خط عربي احترافي

## ⚡ التشغيل المحلي

```bash
git clone https://github.com/almubarmaj23-hub/financial-dashboard-ai
cd financial-dashboard-ai
npm install
npm run dev
# افتح http://localhost:3000
```
